# ti-stats-hook

Titanium CLI hook for stats monitor.

![ti-stats-hook.gif](ti-stats-hook.gif)

## Install

```
$ npm install -g ti-stats-hook
```

## Usage

```
$ ti build -p ios --retina --tall --tistats localhost:8888
```

Please specify your favorite port.
